A production-minded full-stack healthcare management platform built with **Java 21, Spring Boot, React, TypeScript, MySQL, Redis, Docker and Nginx**.

- Spring Security and JWT
- Appointment management
- Medical records and prescriptions
- Billing and invoice generation
- REST APIs and API documentation
- Redis caching and live notifications
- Docker-based deployment
- GitHub Actions continuous integration
